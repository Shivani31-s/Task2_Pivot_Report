# Task 2: Pivot Report Analysis

## Description
This repository contains Task 2 of the Excel Sales Analysis project.  
It includes:
- Cleaned sales data
- Pivot tables
- Pivot chart
- Insights derived from data analysis

## Files
1. **Pivot_Report.xlsx** - Excel file with:
   - Pivot tables
   - Pivot chart
   - Slicers
   - Insights sheet
2. **Pivot_Report.pdf** - PDF export of the Pivot table + Pivot chart for submission

## Insights
1. Office Supplies category generates the highest total sales.
2. Central and West regions contribute the majority of overall revenue.
3. Consumer segment shows consistent performance across regions.
4. Some categories show lower profit margins despite high sales.
5. Discounts negatively impact profit margins in certain regions.

## Instructions
- Open `Pivot_Report.xlsx` to explore data, pivot tables, and chart.
- Use slicers to filter by Region, Segment, or Category.
- PDF is ready for submission to the evaluator.
